Comentar artefatos



  • Alterar Comentário


A partir da Lista de Comentários, clique no ícone  :


Tela Comentar - Ícone Alterar comentário






A tela Alterar Comentário é apresentada em modo de edição para que o texto possa ser alterado:


                                Tela Alterar Comentário





Caso queira desistir, clique no botão . Senão, após a alteração, clique no botão   para salvar as alterações e o sistema apresentará uma mensagem de sucesso:


Mensagem de sucesso após alteração do comentário





Ao clicar no botão Cancelar ou Salvar o sistema retorna para a tela com a Lista de Comentários.










IMPORTANTE!!




Lembrete:

         Os comentários somente poderão ser alterados até que o artefato (documento ou processo) seja tramitado. Após o trâmite essa ação não será mais permitida!
