Despachar artefatos



  • Alterar Despacho



A partir da Lista de Despachos, clique no ícone :


Ícone Alterar Despacho





Uma mensagem de confirmação é apresentada. Ao clicar no botão o sistema apresenta a tela para alteração do despacho. Caso contrário, se clicar no botão o sistema retorna para a tela com a Lista de Despachos.


                                                               Mensagem de confirmação para edição




Ao clicar no botão a tela Alterar Despacho é apresentada em modo de edição para que o texto possa ser alterado:


                  Despacho em modo de edição





Caso queira desistir, clique no botão . Senão, após a alteração, clique no botão   para salvar as alterações e o sistema apresentará uma mensagem de sucesso:


                 Mensagem de sucesso após alteração do despacho




Ao clicar no botão Cancelar ou Salvar o sistema retorna para a tela com a Lista de Despachos.








IMPORTANTE!!




Lembrete:

         Os despachos somente poderão ser alterados até que o artefato (documento ou processo) seja tramitado. Após o trâmite essa ação não será mais permitida!
