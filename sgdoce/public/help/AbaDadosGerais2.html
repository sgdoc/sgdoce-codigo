Cadastrar Processo



  • Aba Dados Gerais



Após a validação do processo o sistema apresenta a tela Cadastrar Processo exibindo a aba Dados Gerais. Observe que alguns campos são obrigatórios (Veja Nota 1):


                                          Tela Cadastrar Processo - Aba Dados Gerais







Informações para cadastro de processo - Aba Dados Gerais:


Âmbito:

De qual esfera o processo é oriundo.

Número do Processo:

Número do processo.

Grau de Acesso:

Grau de acesso que permite ou não a visualização do processo.

Data de Autuação:

Data que consta no processo como sendo a data de sua "criação".





Preenchendo a aba Dados Gerais do cadastro de processo!!


      • O campo Âmbito apresenta a esfera de procedência do processo, sendo exibido automaticamente e desabilitado, ou seja, não é permitida a alteração do seu valor.



      • O campo Número do Processo apresenta o número de processo validado, sendo exibido automaticamente e desabilitado, ou seja, não é permitida a alteração do seu valor.



      • O campo Grau de Acesso possui uma lista com duas opções: Público ou Sigiloso. Selecione a opção desejada para que o valor seja apresentado no campo Grau de Acesso:


                                           Selecionando o grau de acesso do processo na lista para preenchimento do campo Grau de Acesso





      • O campo Data de Autuação já é apresentado preenchido com a data atual, no entanto, se desejar alterar a data, basta clicar no ícone para ser exibido um calendário e utilizá-lo para o preenchimento do campo (Veja Nota 2):


                                                 Clicando no ícone Calendário



                                                    Selecionando a data para preenchimento do campo Data de Autuação

Informações para cadastro de processo - Aba Dados Gerais:


Procedência Externa

Procedência:

Procedência do processo a ser cadastrado: externa.

Tipo de Pessoa:

Tipo da pessoa de acordo com a procedência do processo.

Nacionalidade Brasileira:

Nacionalidade da pessoa física de onde se origina o processo: brasileiro ou estrangeiro.

Nome:

Nome da pessoa física, pessoa jurídica ou Organização externa da procedência do processo (dependendo do tipo de pessoa selecionado) .

CPF:

CPF da pessoa física de onde procede o processo.

Razão Social:

Razão Social da pessoa jurídica de onde procede o processo.

CNPJ:

CNPJ da pessoa jurídica de onde se origina o processo.

Nº do Passaporte:

Número do passaporte da pessoa física estrangeira de onde se origina o processo.

IMPORTANTE!!




Nota 1:


Os campos obrigatórios são indicados pelo sinal de asterisco (*) na cor verde ao lado do nome. Quando algum ou nenhum campo obrigatório for preenchido, o sistema apresenta uma mensagem de erro: "Campos de preenchimento obrigatório não foram preenchidos.".


                                                                   

 Comportamento da tela de Cadastro de Processo - Aba Dados Gerais, após clicar no botão Próxima sem preencher os campos obrigatórios.







Nota 2:


Nos campos de Data, ao clicar no ícone o sistema apresenta um calendário:


                                                                                                Calendário




O calendário é exibido apresentando o mês e ano atuais , além do dia corrente . Utilize as setinhas para navegar entre os meses (anteriores e posteriores) e clique no dia desejado para que o campo de data seja preenchido:


Preenchimento do campo de Data







Nota 3:


Os campos obrigatórios são indicados pelo sinal de asterisco (*) na cor verde ao lado do nome. Quando algum ou nenhum campo obrigatório for preenchido, o sistema apresenta uma mensagem de erro: "Campos de preenchimento obrigatório não foram preenchidos.".




 Comportamento da tela de Cadastro de Processo - Aba Dados Gerais - Procedência Interna, após clicar no botão Próxima sem preencher os campos obrigatórios.







Nota 4:


Os campos obrigatórios são indicados pelo sinal de asterisco (*) na cor verde ao lado do nome. Quando algum ou nenhum campo obrigatório for preenchido, o sistema apresenta uma mensagem de erro: "Campos de preenchimento obrigatório não foram preenchidos.".




 Comportamento da tela de Cadastro de Processo - Aba Dados Gerais - Procedência Externa, após clicar no botão Próxima sem preencher os campos obrigatórios.







Nota 5:


A única diferença que existe no preenchimento do cadastro quando se tratar de Brasileiro ou Estrangeiro é que na primeira opção, ou seja, ao selecionar Nacionalidade Brasileira como Sim, o sistema apresenta o campo CPF. No entanto, tratando-se da segunda opção, ou seja, ao selecionar Nacionalidade Brasileira como Não, o sistema apresenta o campo Passaporte em vez do campo CPF (veja as imagens abaixo):



  Selecionando a opção Sim no campo Nacionalidade Brasileira: o campo CPF é apresentado logo abaixo do campo Nome




  Selecionando a opção Não no campo Nacionalidade Brasileira: o campo Nº do Passaporte é apresentado logo abaixo do campo Nome

Nota 7:


A mesma situação acontece para estrangeiro, ou seja:

        • Se o estrangeiro não estiver cadastrado, deverá ser feito conforme Nota 6;
        • Ao informar o Nome do estrangeiro, o sistema retorna automaticamente o número do seu passaporte;
        • É possível inserir o número do passaporte para que o sistema retorne automaticamente o nome do estrangeiro;
        • Na situação em que seja informado o número do passaporte e este não for encontrado pelo sistema, deve ser cadastrado conforme Nota 6.
