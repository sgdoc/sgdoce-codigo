Despachar artefatos



  • Cadastrar Despacho



Para incluir um novo despacho, clique no ícone  na Lista de Despachos:


  Tela Detalhar Despachos - Ícone Adicionar novo despacho





O sistema apresenta a tela Cadastrar Despacho. Observe que todos os campos são obrigatórios (Veja Nota):


                                                      Tela Cadastrar Despacho - Despachar documento ou processo




Informações para despacho:


Destinatário

Nome da Unidade de destino do despacho

Despacho

Texto do despacho

Assinatura:

Nome do assinante do artefato (documento/processo)

Cargo:

Nome do cargo do assinante





Preenchendo o despacho!!


      • O campo Destinatário é autocomplete, ou seja, à medida que o texto vai sendo digitado dentro do campo, uma lista correspondente à digitação é apresentada logo abaixo. Essa ação ocorre a partir do terceiro caracter informado e tem por objetivo facilitar a busca da informação. Após visualizar o item desejado na lista, clique em cima dele para que o nome da Unidade seja apresentado no campo Destinatário:



                                                                         Selecionando uma Unidade na lista para preencher campo Destinatário




      • O campo Despacho possibilita preencher um texto de até 1000 caracteres. Abaixo desse campo existe um “contador de caracteres” que informa quantos caracteres já foram preenchidos e quantos ainda podem ser informados. À medida que o texto vai sendo digitado os valores desse “contador” vão sendo atualizados.



                                                                   Preenchendo o campo Despacho




      • O campo Assinatura é autocomplete, ou seja, à medida que o texto vai sendo digitado dentro do campo, uma lista correspondente à digitação é apresentada logo abaixo. Essa ação ocorre a partir do terceiro caracter informado e tem por objetivo facilitar a busca da informação. Após visualizar o item desejado na lista, clique em cima dele para que o nome do Assinante seja apresentado no campo Assinatura.


                                                                         Selecionando um nome na lista para preencher campo Assinatura




      • O campo Cargo possui uma lista de opções. Selecione a opção correspondente para que o valor seja apresentado no campo Cargo:


                                                      Selecionando um cargo na lista para preenchimento do campo Cargo



      • Após preencher todos os campos clique no botão . O sistema apresentará uma mensagem de sucesso e o despacho será apresentado na lista de despachos. Caso contrário, se desistir de despachar, clique no botão :


                                                  Tela Cadastrar Documento - Despacho preenchido




      • Ao clicar no botão Cancelar ou Salvar o sistema retorna para a tela com a Lista de Despachos:


Mensagem de sucesso após cadastro de despacho









IMPORTANTE!!




Lembrete:

         Quando o despacho for de saída da Unidade ou do Instituto será necessária a impressão da folha de despacho, além da assinatura física do responsável pela área!






Nota:


Os campos obrigatórios são indicados pelo sinal de asterisco (*) na cor verde ao lado do nome. Quando algum ou nenhum campo obrigatório for preenchido, o sistema apresenta uma mensagem de erro: "Campos de preenchimento obrigatório não foram preenchidos.".


                                                                           Comportamento da tela de Despacho ao tentar salvar sem preencher os campos obrigatórios.
