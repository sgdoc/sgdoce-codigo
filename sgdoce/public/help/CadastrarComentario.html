Comentar artefatos



  • Cadastrar Comentário


Para incluir um novo comentário, clique no ícone  na Lista de Comentários:


   Adicionar novo comentário





O sistema apresenta a tela Cadastrar Comentário. Observe que o campo Comentário é obrigatório (Veja Nota):


     Comentar documento ou processo








Preenchendo o comentário!!


      • O campo Comentário possibilita preencher um texto de até 2000 caracteres. Abaixo desse campo existe um “contador de caracteres” que informa quantos caracteres já foram preenchidos e quantos ainda podem ser informados. À medida que o texto vai sendo digitado os valores desse “contador” vão sendo atualizados.



                   Preenchendo o campo comentário




      • Após preencher o campo Comentário clique no botão . O sistema apresentará uma mensagem de sucesso e o comentário será apresentado na lista de comentários. Caso contrário, se desistir de comentar, clique no botão :


Tela Comentar após o cadastro - Mensagem de sucesso



Ao clicar no botão Cancelar ou Salvar o sistema retorna para a tela com a Lista de Comentários.











IMPORTANTE!!




Nota:


Os campos obrigatórios são indicados pelo sinal de asterisco (*) na cor verde ao lado do nome. O campo Comentário, portanto, é obrigatório, ou seja, se houver uma tentativa de salvar o comentário em branco, o sistema apresenta uma mensagem de erro: "Campos de preenchimento obrigatório não foram preenchidos.".



                                                     Comportamento da tela de Comentário ao tentar salvar em branco.
