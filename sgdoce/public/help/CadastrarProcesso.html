Cadastrar Processo



A funcionalidade de Cadastrar Processo é utilizada para cadastrar todos os processos recebidos. Processos autuados no SGDoc não precisam de cadastro, pois no momento da autuação o cadastro é realizado automaticamente.


Para cadastrar um processo selecione a opção Processo no menu do sistema SGDoc:



                                                                                                    Menu do sistema SGDoc





O cadastro do processo é realizado em duas partes: Validar Processo e Cadastrar Processo. Primeiramente é feita a validação do processo de acordo com o Âmbito: Federal, Estadual, Municipal ou Judicial:


                                                              Tela Validar Digital





Após a validação do processo a tela Cadastrar Processo é apresentada exibindo 5 abas: Dados Gerais, Volumes, Interessados, Assunto e Referências:


                                                  Tela Cadastrar Processo





Após o cadastro, o Processo estará disponível na Área de Trabalho - Minha, do usuário que o cadastrou.







Formas de "navegar" pelas abas do cadastro de processo!




                                          Barra de Navegação da Tela Cadastro de Processo




            • Botão : Utilize o botão Anterior para retornar às abas anteriores. Quando o cadastro de processo estiver exibindo a primeira aba, o botão ficará desabilitado:              




            • Botão : Utilize o botão Próxima para avançar para as abas posteriores. Quando o cadastro de processo estiver exibindo a última aba, o botão ficará desabilitado:    




            • Botão : Utilize o botão Concluir para finalizar o cadastro do processo. Nas abas Dados Gerais e Origem/Destino o botão é apresentado desabilitado . Somente será habilitado a partir da aba Assunto.





            • Botão : Utilize o botão Cancelar para fechar o cadastro de processo, cancelando os dados informados. Ao clicar no botão o sistema apresenta a mensagem:




Ao clicar no botão o sistema retorna para a tela de Cadastrar Processo, caso contrário, ao clicar no botão o sistema retorna para a tela de Validar Processo.


 









IMPORTANTE!!



Lembrete 1:

         Deve-se preencher todos os campos obrigatórios antes de passar para a aba seguinte (antes de clicar no botão Próxima). Caso contrário o sistema apresenta mensagem de erro por haver campos obrigatórios a serem preenchidos!



Lembrete 2:

         O cadastro de processo somente pode ser finalizado (clicar no botão Concluir) após terem sido preenchidos todos os campos obrigatórios do cadastro de processo. Caso contrário o sistema apresenta mensagem de erro por haver pendências de obrigatoriedade!
