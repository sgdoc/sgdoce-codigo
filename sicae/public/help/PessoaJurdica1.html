Cadastrar Pessoa Jurídica



Dados Básicos



Informe os dados, observando que os campos com asterísco (*) são de preenchimento obrigatório.



  • CNPJ: digite o CNPJ (Cadastro Nacional de Pessoa Jurídica) válido;
  • Razão Social : informe o nome da razão social da pessoa jurídica;
  • Nome Fantasia : informe o nome fantasia;
  • Natureza Jurídica : selecione a natureza jurídica;
  • Classificação da Natureza Jurídica : selecione a classificação de acordo com a natureza jurídica;
  • Tipo de Estabelecimento : selecione o tipo de estabelecimento (Filial ou Matriz).


Após informar os dados clique no botão desejado:

Salve as informações editadas, acionando a opção;


Para Salvar e seguir para próxima tela, acione a opção ;


Para Concluir o cadastro, acione a opção e confirme os dados informados;


Para não registrar os dados informados, acione a opçãoo sistema retorna para a tela "Pesquisar Pessoa".
