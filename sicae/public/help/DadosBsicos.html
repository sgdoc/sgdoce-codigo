Cadastrar Pessoa Física



Dados Básicos



Informe os dados, observando que os campos com asterísco (*) são de preenchimento obrigatório.



  • CPF: digite o CPF (Cadastro de Pessoa Física) válido;
  • Nome: informe o nome completo;
  • Data de Nascimento: informe a data de nascimento ou selecione acionando o botão ;
  • Estado Civil: informe o estado civil;
  • Nacionalidade Brasileira?:  informe a opção "Sim" se for Brasileira(o), para as demais informe a opção "Não";
  • Estado Naturalidade: selecione o Estado onde nasceu;
  • Município Naturalidade: selecione o Município referente ao Estado da Naturalidade;
  • Sexo: selecione o sexo;
  • Nome do Pai: informe o nome completo do Pai;
  • Nome da Mãe: informe o nome completo da Mãe.


Após informar os dados clique no botão desejado:


Salve as informações, acionando a opção;


Para Salvar e seguir para próxima tela, acione a opção ;


Para Salvar e Concluir o cadastro, acione a opção e confirme os dados informados;


Para cancelar o cadastro, acione a opçãoo sistema retorna para a tela "Pesquisar Pessoa".
