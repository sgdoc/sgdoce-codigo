Cadastrar Usuário Interno

Para cadastrar Usuário Interno, acione a opção e preencha os campos solicitados observando que os campos com asterísco (*) são de preenchimento obrigatório.


Para salvar os dados informados, acione a opção ;


Para desistir do cadastro, acione a opção o sistema retorna para a tela "Pesquisar Usuário Interno".


Obs.: O usuário a ser cadastrado deve existir no cadastro de pessoa física, e o mesmo deve possuir e-mail e telefone institucional.
